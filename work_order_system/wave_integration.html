React Props:
  • theme: 'ocean', 'sunset', 'forest', 'purple', 'mint'
  • intensity: 'low', 'medium', 'high'
  • speed: 'slow', 'normal', 'fast'
  • layers: Number of wave layers (1-6)
  • showControls: Show theme switcher
